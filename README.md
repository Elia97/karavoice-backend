> Le prenotazioni possono includere note opzionali, numero di partecipanti, e uno stato personalizzabile (pending, confirmed, cancelled)

#### GET `/api/bookings`

- `200 Ok` ‚Üí Lista completa prenotazioni
- `500 Internal Server Error` ‚Üí Errore generico

#### GET `/api/bookings/user`

- `200 Ok` ‚Üí Lista prenotazioni dell'utente
- `404 Not Found` ‚Üí Nessuna prenotazioni trovata
- `500 Internal Server Error` ‚Üí Errore generico

#### GET `/api/bookings/event/:eventId`

- `200 Ok` ‚Üí Lista prenotazioni per evento
- `404 Not Found` ‚Üí Nessuna prenotazioni trovata
- `500 Internal Server Error` ‚Üí Errore generico

#### GET `/api/bookings/:id`

- `200 Ok` ‚Üí Prenotazione trovata
- `404 Not Found` ‚Üí Prenotazione non trovata
- `500 Internal Server Error` ‚Üí Errore generico

#### POST `/api/bookings`

```json
{
  "user_id": "UUID",
  "event_id": "UUID",
  "status": "pending | confirmed | cancelled", // default: pending
  "participants": 3,
  "notes": "Note aggiuntive"
}
```

- `201 Created` ‚Üí Prenotazione creata
- `500 Internal Server Error` ‚Üí Errore generico

#### PUT `/api/bookings/:id/status`

```json
{
  "status": "confirmed"
}
```

- `200 Ok` ‚Üí Stato aggiornato
- `404 Not Found` ‚Üí Prenotazione non trovata
- `500 Internal Server Error` ‚Üí Errore generico

#### DELETE `/api/bookings/:id`

- `204 No Content` ‚Üí Prenotazione eliminata
- `404 Not Found` ‚Üí Prenotazione non trovata
- `500 Internal Server Error` ‚Üí Errore generico
